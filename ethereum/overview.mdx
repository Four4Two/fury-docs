Learn how to build fast, scalable, and secure dApps on Highbury. Highbury is EVM compatible.

## What is Highbury EVM?

Highbury is a lighting fast blockchain built with the Cosmos SDK that includes Highbury EVM, an EVM compatible smart contract execution platform. Highbury's best-in-class Tendermint based BFT consensus engine allows it to be quick, cheap, and secure.

## Key features

### Speed

With single block finality and unrivalled scalability, Tendermint Consensus enables Highbury to support the transaction needs of thousands of protocols and millions of users.

### Interoperability

Highbury is connected to the 30 chains and \$60B+ of the Cosmos ecosystem via the IBC protocol.

### Smart contract support

Highbury EVM is fully compatible with Ethereum, offering an execution environment that empowers Solidity developers and their dApps to benefit from the scalability and security of the Highbury Network.

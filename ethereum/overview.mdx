## What is Blackfury EVM?

Blackfury is a lighting fast blockchain built with the Cosmos SDK that includes Blackfury EVM, an EVM compatible smart contract execution platform. Blackfury's best-in-class Tendermint based BFT consensus engine allows it to be quick, cheap, and secure.

## Key features

### Speed

With single block finality and unrivalled scalability, Tendermint Consensus enables Blackfury to support the transaction needs of thousands of protocols and millions of users.

### Interoperability

Blackfury is connected to the 30 chains and \$60B+ of the Cosmos ecosystem via the IBC protocol.

### Smart contract support

Blackfury EVM is fully compatible with Ethereum, offering an execution environment that empowers Solidity developers and their dApps to benefit from the scalability and security of the Blackfury Network.

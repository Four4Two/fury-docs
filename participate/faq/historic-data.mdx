As of the Fury 10 release, all upgrades are done as in-place software upgrades.
The chain id no longer updates and the block height is never reset to 0, as was the case with all
upgrades prior to Fury 10. Historical data for all upgrades since Fury 10 are available on the
mainnet data node endpoints. Some data may need to be queried with the historic verison of the Fury CLI.

Note that the Fury 5 upgrade was reverted. Those data are also available: [API](https://api.fury-5.archives.fury.black/node_info) & [RPC](https://rpc.fury-5.archives.fury.black/status)

# EVM Overview

Learn how to build fast, scalable, and secure dApps on Fury. Fury is EVM compatible.

## What is Fury EVM?

Fury is a lighting fast blockchain built with the Cosmos SDK that includes Fury EVM, an EVM compatible smart contract execution platform. Fury's best-in-class Tendermint based BFT consensus engine allows Fury to be quick, cheap, and secure.

## Key features

### Speed

With single block finality and unrivalled scalability, Tendermint Consensus enables Fury to support the transaction needs of thousands of protocols and millions of users.

### Interoperability

Fury is connected to the 30 chains and \$60B+ of the Cosmos ecosystem via the IBC protocol.

### Smart contract support

Fury EVM is fully compatible with Ethereum, offering an execution environment that empowers Solidity developers and their dApps to benefit from the scalability and security of the Blackfury Network.
